# webpage
Responsive Webpage Project

 Overview

This project is a modern, fully responsive website built using HTML, CSS, and JavaScript.
It demonstrates clean UI design and includes sections like hero, about, services, team, works, and clients.

 Features

-  Fully responsive design (mobile, tablet, desktop)
-  Clean and modern UI/UX
-  Navigation bar with toggle menu
-  Animated counters
-  Team section with hover effects
-  Services and works sections
-  Interactive client section (click highlight)
-  Smooth user experience

 Technologies Used

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript
